# **App Name**: Ishan's Digital Portfolio

## Core Features:

- Profile Summary: Dynamic display of Ishan's professional summary, highlighting key skills and experiences.  Automatically updates based on linkedIn profile. Use an AI tool to analyze the profile and update summary if needed.
- Interactive Skills Showcase: Visually appealing and interactive display of skills categorized into Frontend, Backend, Databases, and Tools, enhancing user engagement.
- Dynamic Project Portfolio: A filterable and detailed showcase of Ishan's projects with descriptions, tech stacks, and links to GitHub repositories and live demos.
- Experience Timeline: Timeline displaying Ishan's experiences, activities (Hackathons, Code for Change), training, and workshops.
- Contact Form: A functional contact form with validation, allowing visitors to directly send messages to Ishan.
- SEO Optimization: Implement best practices for search engine optimization, including meta tags and schema markup, to improve website visibility.

## Style Guidelines:

- Primary color: Soft blue (#89B4FA) for a professional and calming feel.
- Background color: Off-white (#F5F5F5) for a clean and modern look.
- Accent color: Teal green (#32CD32) for highlights and call-to-action buttons, adding a touch of vibrancy.
- Body and headline font: 'Inter' sans-serif, for a modern and neutral aesthetic; one font can serve both purposes.
- Note: currently only Google Fonts are supported.
- Use simple, line-based icons from a library like FontAwesome or Remix Icon for a clean, modern feel.
- Utilize a grid-based layout with consistent spacing and padding for a balanced and professional design.
- Implement subtle transitions and hover effects using CSS or a library like Framer Motion to enhance user experience.
